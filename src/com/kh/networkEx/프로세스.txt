프로그램 실행시 운영체제에서 메모리 할당받아 실행
각 프로세스는 독립된 메모리 가짐
다른 프로세스와 메모리 공유 안함

프로세스는 스레드를 포함한다
하나의 프로세스는 최소 하나의 스레드(작업) 포합
스레드는 프로세스(작업공간)에서 실행되는 프로그램의 실행흐름 담당

하나의 프로세스 안에서 여러 스레드가 동시에 각자 작업실행, 각 스레드는 작업공유 가능, 작업 동시수행 가능
프로세스는 독립실행되는 프로그램의 인스턴스, 메모리 가짐
스레드는 프로세스 내 작업으로 프로세스 자원 공유, 동시실행 가능

메모장 크롬 그림판 -> 멀티 프로세스 상태, 각 프로세스가 하나의 작업시 단일스레드, 프로세스가 여러작업 실행시 멀티스레드 상태
메모장 -> 단일 프로세스 상태, 프로세스가 하나의 작업시 단일스레드
